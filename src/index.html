Security Header Test


Strict-Transport-Security

Used to ensure that the browser is using a secure connection.

CSU Recommendation: Required

Optimal Header = max-age=31536000; includeSubDomains; preload

Content-Security-Policy

Controls what scripts can be loaded on the page.

CSU Recommendation: Optional

Optimal Header = default-src ‘self’; script-src *; font-src *; style-src *

X-Frame-Options

Utilized to prevent UI Redressing attacks that stem from clickjacking.

CSU Recommendation: Recommended

Optimal Header = SAMEORIGIN

X-XSS-Protection

Detects and prevents malicious HTML input.

CSU Recommendation: Recommended

Optimal Header = 1; mode=block

X-Content-Type-Options

A MIME-sniffing utility that informs the browser what to do with file types.

CSU Recommendation: Required

Optimal Header = nosniff

X-Download-Options

Prevents html files from running and forces manual download.

CSU Recommendation: Optional

Optimal Header = noopen

Referrer-Policy

Allows users to track where they came from.

CSU Recommendation: Optional

Optimal Header = strict-origin-when-cross-origin

Feature-Policy

Used to enable or disable certain features on the browser.

CSU Recommendation: Optional

Optimal Header = microphone‘none’; payment‘none’; sync-xhr ‘self

Public-Key-Pins

Allows websites to resist fraudulent digital certificates.

CSU Recommendation: Optional

Optimal Header = This header is extremely customizable, so the optimal header example can change significantly.
